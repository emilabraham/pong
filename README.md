# Pong

## Description

A buggy recreation of the first video game.

## Achievement ideas

1. Own Goal: Your paddle is the last thing hit before scoring on yourself.
2. Skunk: End the game with a skunk score: 7-0.
3. First Victory: Be the first to get to 11 points and win the game.
4. Longest Rally: Bounce off the paddle 20 times before scoring a point.
